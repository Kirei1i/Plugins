# Plugins

个人维护的Dalamud插件仓库，主要用于汉化和优化。

**本仓库所有插件禁止各种形式的倒卖！**

## 仓库地址

```
https://raw.githubusercontent.com/Kirei1i/Plugins/main/pluginmaster.json
```

## 使用方法

1. 打开游戏内的Dalamud设置
2. 进入「Experimental」选项卡
3. 在「Custom Plugin Repositories」中添加上方仓库地址
4. 点击「+」按钮保存
5. 在插件列表中即可找到本仓库的插件

## 包含插件

| 插件 | 说明 |
|------|------|
| PartyIcons | 队伍名牌图标显示（中文汉化版） |
